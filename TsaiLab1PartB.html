How do you use <ol>; <ul>; <dt>?

<ol>:

This tag is used to create an ordered list, which displays items in a numbered sequence. Within the <ol>, you enclose the list items in <li> and </li>. Then you close it up with a </ol>.
For example:
<ol>
<li>hello</li>
<li>hi</li>
<li>hallo</li>
</ol>

<ul>:

This tag is used to create an unordered list, which displays items in an unordered sequence using bullet points. Within the <ul>, you enclose the list items in <li> and </li>. Then you close it up with a </ul>.
For example:
<ul>
<li>chinchilla</li>
<li>kangaroo</li>
<li>panda</li>
</ul>

<dt>:

This tag is used to define a discription term. It is always within/a direct child of a discription list (<dl>). The <dt> would then be followed by a <dd> (description details) which is the discription definition.
For example:
<dl>
<dt>What is a sphere?</dt>
<dd>A sphere is a perfectly round, three-dimensional object where every point on its surface is equidistant from a central point.</dd>
</dl>

Difference between text editor and word processor?

A text editor is best for coding and simple scripts because it works with plain, unformatted text. On the other hand, word processors are used for creating formatted documents such as essays and reports, using features such as bolding, italics, images, and spell-check.

What is the client-server model?

The client-server model is a network where clients, such as computers or other devices, send requests for services or resources to a centralized server. The server then processes these requests and provides the appropriate responses. This model is widely utilized in internet and network communications which enables efficient resource sharing and data exchanges.

Provide a description and definition for the following network protocols: FTP, HTTP, TCP/IP

FTP:

FTP (File Transfer Protocal) is a standard network protocol used for transferring files between a client and a server. It allows users to upload, download, and manage files remotely in a safe and organized way.

HTTP:

HTTP (Hypertext Transfer Protocol) is the system that allows web browsers and servers to communicate. It creates the foundation for how data is transferred and displayed online.

TCP/IP:

TCP/IP (Transmission Control Protocol/Internet Protocol) is a set of rules that control how data is transmitted over the internet and other networks to ensure reliable and efficient communication. It's the foundational protocol that allows devices to connect and share information globally.
